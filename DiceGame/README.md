## 🎲 Dice Game

A simple browser-based dice game where two players roll a die. The higher number wins!

---

![Dice Game Screenshot](DiceGame/img.png)

### 🚀 Features
- Random dice rolls
- Visual dice dot patterns
- Win/draw announcement
- Clean and responsive layout

### 🛠 Built With
- HTML
- CSS
- JavaScript
